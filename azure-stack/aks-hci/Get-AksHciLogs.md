# Get-AksHciLogs

## SYNOPSIS
Create a zipped folder with logs from all your pods. This command will create an output zipped folder called `akshcilogs` in the path `C:\wssd\akshcilogs`.

## SYNTAX

```powershell
Get-AksHciLogs
```

## DESCRIPTION
Create a zipped folder with logs from all your pods. This command will create an output zipped folder called `akshcilogs` in the path `C:\wssd\akshcilogs`.

## EXAMPLES

### Example
```powershell
PS C:\> Get-AksHciLogs
```

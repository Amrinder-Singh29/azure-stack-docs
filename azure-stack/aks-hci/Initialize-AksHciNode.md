# Initialize-AksHciNode

## SYNOPSIS
Run checks on every physical node to see if all requirements are satisfied to install Azure Kubernetes Service on Azure Stack HCI.

## SYNTAX

```powershell
Initialize-AksHciNode
```

## DESCRIPTION
Run checks on every physical node to see if all requirements are satisfied to install Azure Kubernetes Service on Azure Stack HCI.

## EXAMPLES

### Example
```powershell
PS C:\> Initialize-AksHciNode
```

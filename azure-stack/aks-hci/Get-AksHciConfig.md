# Get-AksHciConfig

## SYNOPSIS
List the current configuration settings for the Azure Kubernetes Service host.

## SYNTAX

```powershell
Get-AksHciConfig
```

## DESCRIPTION
List the current configuration settings for the Azure Kubernetes Service host.

## EXAMPLES

PS C:\> Get-AksHciConfig
